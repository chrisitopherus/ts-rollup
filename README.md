# ts-rollup

## Get Started

```zsh
git clone https://github.com/chrisitopherus/ts-rollup.git <your-project-name>

# change directory to your project
cd <your-project-name>

# install all dependencies.
npm i
npm install

# build
npm run build

# test
npm test
npm run test

# for more check scripts at the package json :)
```
